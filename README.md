# My Student Grade Tracker

A simple command-line grade tracker.

## What it does
- Add grades
- View all grades
- Track grades
- Delete grades

## How to run
1. Activate virtual environment: `source venv/bin/activate`
2. Run app: `python src/main.py`

## Requirements
- Python 3.8 or higher
